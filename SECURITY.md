# Security Policy

## Supported Versions

本仓库为个人 Profile 仓库，目前仅 `main` 分支受支持。

| Branch  | Supported          |
| ------- | ------------------ |
| `main`  | :white_check_mark: |
| 其他    | :x:                |

## Reporting a Vulnerability

如发现本仓库存在安全漏洞 (例如 README 中嵌入了恶意链接、CI workflow 注入风险等),
请通过以下渠道私密披露:

- **GitHub Security Advisories**: <https://github.com/xfengyin/xfengyin/security/advisories/new>
- **响应 SLA**: 7 个工作日内确认收到

请勿在公开 Issue 中直接披露未修复的漏洞细节。

## Scope

本仓库的内容范围:

- `README.md` — 个人介绍与项目展示
- `data/*.yml` — 精选项目 / 技术栈数据
- `.github/workflows/*` — CI 工作流
- `docs/ARCHITECTURE.md` — 架构说明

## Out of Scope

- 第三方徽章服务 (shields.io / github-readme-stats) 的可用性
- 个人主页 <https://xfengyin.github.io> 的安全策略
